# Create a PostgreSQL Instance

This is an example of how to create a PostgreSQL instance.

## 1. Create a PostgreSQLInstance manifest

Create an PostgreSQLInstance manifest and deploy it to the cluster. It is a good practice to include it in the application's Helm chart.

Security group and security group rules which allow access to the PostgreSQLInstance from pods are created automatically.

Connection details are stored in a Kubernetes secret `<.metadata.name>-dbadmin` and AWS Secrets Manager secret.

```yaml
# Example PostgreSQLInstance
apiVersion: database.entigo.com/v1alpha1
kind: PostgreSQLInstance
metadata:
  name: example-postgresql
spec:
  allocatedStorage: 20
  engineVersion: '17.2'
  instanceType: 'db.t3.small'

---
# Example PostgreSQLInstance with custom settings
apiVersion: database.entigo.com/v1alpha1
kind: PostgreSQLInstance
metadata:
  name: example-postgresql
spec:
  allocatedStorage: 20
  engineVersion: '17.2'
  instanceType: 'db.t3.small'
  iops: 3000
  multiAZ: true
  parameterGroupName: 'default.postgres17'
  deletionProtection: true
  autoMinorVersionUpgrade: true
  allowMajorVersionUpgrade: false
  backupWindow: '05:00-05:00'
  maintenanceWindow: 'Mon:00:00-Mon:03:00'
```

## 2. Mount PostgreSQLInstance credentials to a container

Connection details are stored in a Kubernetes secret `<.metadata.name>-dbadmin` and AWS Secrets Manager secret.

For more information about Secrets in Kubernetes, see [Kubernetes documentation](https://kubernetes.io/docs/concepts/configuration/secret/).

```yaml
# Example
apiVersion: v1
kind: Pod
metadata:
  name: postgres
spec:
  containers:
    - name: postgres
      image: postgres:alpine
      command: ['sleep', 'infinity']
      env:
        - name: PGHOST
          valueFrom:
            secretKeyRef:
              name: example-postgresql-dbadmin
              key: endpoint
        - name: PGPORT
          valueFrom:
            secretKeyRef:
              name: example-postgresql-dbadmin
              key: port
        - name: PGUSER
          valueFrom:
            secretKeyRef:
              name: example-postgresql-dbadmin
              key: username
        - name: PGPASSWORD
          valueFrom:
            secretKeyRef:
              name: example-postgresql-dbadmin
              key: password
        - name: PGSSLMODE
          value: 'require'
```
