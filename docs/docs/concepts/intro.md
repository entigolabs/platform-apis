# Intro

## Multi-tenancy

Entigo Platform supports multi-tenancy for service provider use cases and organizational scenarios requiring strong isolation at the identity, access, and infrastructure levels.

- [Organization](organization) enables service providers to isolate different customers or independent subsidiaries within a single corporation.
- [Workspace](workspace) provides cloud-provider level isolation, for example, to implement strong resource ownership between different business units or separation between lifecycle environments.
- [Zone](zone) provides a cost-effective approach to implement security isolation for organizations or business units managing multiple data assets.
